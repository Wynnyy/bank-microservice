## Bank application

### Make sure to have installed
* Maven 3.9.1 or later
* JAVA 17 or later

##For run this application:

- must have Mysql which running on 127.0.0.1:3306
- run mvn command bellow when Mysql is running

### Build & Test:

```
mvn clean install
```
